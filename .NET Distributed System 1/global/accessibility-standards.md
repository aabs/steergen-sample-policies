# Accessibility Standards

:::rule id="STD-A11Y-001" severity="error" category="wcag" domain="accessibility"
title: WCAG 2.1 AA Compliance

All UI components shall comply with WCAG 2.1 AA standards.
:::

:::rule id="STD-A11Y-002" severity="error" category="testing" domain="accessibility"
title: Accessibility Testing in CI

The CI pipeline shall include accessibility testing via axe or Lighthouse CI.
:::

:::rule id="STD-A11Y-003" severity="error" category="review" domain="accessibility"
title: Periodic UX Accessibility Reviews

The team shall conduct periodic UX accessibility reviews.
:::
