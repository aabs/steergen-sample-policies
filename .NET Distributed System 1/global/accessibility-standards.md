# Accessibility Standards

:::rule id="STD-A11Y-001" severity="error" category="wcag" domain="accessibility" tags="accessibility, wcag, ui, components, comply, aa"
All UI components shall comply with WCAG 2.1 AA standards.
:::

:::rule id="STD-A11Y-002" severity="error" category="testing" domain="accessibility" tags="accessibility, testing, ci, pipeline, axe, lighthouse"
The CI pipeline shall include accessibility testing via axe or Lighthouse CI.
:::

:::rule id="STD-A11Y-003" severity="error" category="review" domain="accessibility" tags="accessibility, review, conduct, periodic, ux, reviews"
The team shall conduct periodic UX accessibility reviews.
:::
