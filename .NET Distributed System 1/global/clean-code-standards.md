# Clean Code Standards (Cross-Cutting)

:::rule id="STD-CLN-001" severity="error" category="constants" domain="clean-code"
title: Named Constants

The codebase shall not contain magic numbers; named constants shall be used instead.
:::

:::rule id="STD-CLN-002" severity="error" category="code-quality" domain="clean-code"
title: Dead Code Removal

The codebase shall not contain dead or unreachable code.
:::

:::rule id="STD-CLN-003" severity="error" category="readability" domain="clean-code"
title: Code Readability and Maintainability

All code shall be readable, maintainable, and documented.
:::
