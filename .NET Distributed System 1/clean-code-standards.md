# Clean Code Standards (Cross-Cutting)

:::rule id="STD-CLN-001" severity="error" category="constants" domain="clean-code" tags="clean-code, constants, contain, magic, numbers, named"
The codebase shall not contain magic numbers; named constants shall be used instead.
:::

:::rule id="STD-CLN-002" severity="error" category="code-quality" domain="clean-code" tags="clean-code, code-quality, contain, dead, unreachable"
The codebase shall not contain dead or unreachable code.
:::

:::rule id="STD-CLN-003" severity="error" category="readability" domain="clean-code" tags="clean-code, readability, readable, maintainable, documented"
All code shall be readable, maintainable, and documented.
:::
