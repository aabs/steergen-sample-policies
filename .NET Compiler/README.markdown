# .NET Compiler Constitution Files

These documents are a distillation of the steering docs used in the construction of the [Fifth Compiler](https://fifth-lang.org).

For more info on the development of the compiler using Github Spec-Kit, see this [retrospective](https://aabs.wordpress.com/2025/12/05/the-story-of-fifth-so-far/).

To validate this collection:

```shell
steergen validate --global ./.NET\ Compiler

Validation complete: 0 error(s), 0 warning(s).
```
